# 体験談DB 統一テンプレート v2

**更新日**: 2026-03-09
**変更点**: v1 → v2
- 目標文字量を 6,000-8,000字 に設定
- 埋め込みリンクマーカー `[EMBED]` を重要セクションに追加
- 各セクションの文字量ガイドラインを追加
- 肉付け指針（感情描写・サブエピソード・読者への問いかけ）を追加

---

## 共通ルール

### 文字量

| 項目 | 基準 |
|------|------|
| 目標文字量 | 6,000-8,000字（スペックカード・合成ソースメモ除く） |
| 読了時間の目安 | 電車内10-15分 |
| 最低文字量 | 5,500字（これ以下はリジェクト） |

### 埋め込みリンク方針: ポイント埋め込み

- `[EMBED]` マーカーのあるセクションのみ、1-2個の実在リンクを配置
- リンク先は**実在する**YouTube動画、ブログ記事、店舗、サービス、書籍等
- リンクの前後に**体験文脈**を必ず書く（「このリンクを見て何が変わったか」のストーリー）
- noteの埋め込み対応フォーマットを使用

### 肉付け指針

各セクションで以下を意識する：

1. **感情の具体化**: 「辛かった」ではなく「駅のホームで一人スマホを見ている自分がふと嫌になった」
2. **サブエピソード**: 友人との会話、SNSで見た投稿への反応、家族の言葉など
3. **内面の葛藤**: 「やめようと思った瞬間」「それでも続けた理由」の対比
4. **数字の文脈**: 数字を出したら「それが生活にどう影響したか」を必ず添える
5. **読者への問いかけ**: セクション末尾に共感ポイントや問いかけを適宜配置

#### 体験記セクション構成

```
## タイトル: [パターン名]

### きっかけ — なぜ婚活を始めたのか
目安: 800-1,200字
ポイント: 日常の具体的な場面から始める。「焦り」の感情を丁寧に描写。

### 迷走期 — 最初にぶつかった壁
目安: 1,200-1,800字
ポイント: 3-5人の具体的なエピソード。「消費される感覚」等の心理変化を描く。
[EMBED] 婚活疲れ関連のリソース（共感記事、休息法の動画等）

### 転機 — 何が変わったのか
目安: 1,000-1,500字
ポイント: 内面の気づき + 具体的な行動変容。何がきっかけで考えが変わったか。
[EMBED] 転機となったリソース（自分磨き動画、マインドセット記事、本等）

### 出会い — 「この人かも」と思った瞬間
目安: 1,000-1,500字
ポイント: 五感を使った場面描写。「ドキドキ」ではなく「安心感」の発見。

### 決断 — 交際/成婚に至るまで
目安: 600-1,000字
ポイント: 迷いと決断の対比。相手の視点も入れる。

### 振り返り — 過去の自分に伝えたいこと
目安: 800-1,200字
ポイント: 5つ前後の具体的な教訓。読者が自分に当てはめられる形で。
```

## 合成ソース・メモ（内部用・非公開）

全パターン共通で記事末尾に付与。公開時は削除する。

```
## 合成ソース・メモ（内部用・非公開）

本パターンは以下の共通テーマを合成:
- [ソース1の概要]（出典）
- [ソース2の概要]（出典）
- ...

属性は全て帯域化済み。エピソードは複数ソースから合成した架空のもの。
文章のコピペは一切なし。
```

## Neo4j連携

各パターンは以下のノード/リレーションで管理:

```
(:ExperiencePattern {pattern_id, theme, title, wordcount_target, wordcount_current, status})
  -[:BELONGS_TO]-> (:CandidateTheme)
  -[:EMBEDS {section}]-> (:EmbeddableResource {resource_id, resource_type, url, title, embed_section, experience_context})

(:Memory:project {name: '副業プロジェクト'})
  -[:HAS_PATTERN]-> (:ExperiencePattern)
```

### ID命名規則

| ノード | ID形式 | 例 |
|--------|--------|-----|
| ExperiencePattern | `exp-{theme}-{sequential}` | `exp-konkatsu-001` |
| EmbeddableResource | `res-{type}-{sequential}` | `res-youtube-001` |
