# 副業プロジェクト：レビュー記事のジャンル拡大戦略 (2026-03-09)

「投資本レビュー」のフレームワーク（3分核心＋2冊比較）を、より広い層にリーチするために多角化する。

## 1. 拡大ジャンルの定義

### A. ビジネス・仕事術・自己啓発
*   **ターゲット**: タイパ重視のビジネスパーソン、自己成長を求める層。
*   **切り口**: 「投資家視点で読む自己啓発」。自己投資としてのリターン（年利換算など）を評価軸に加える。
*   **役割**: 安定したストック資産。note内での検索流入や高単価マガジンへの繋ぎ。
*   **比較例**: 『エッセンシャル思考』vs『限りある時間の使い方』

### B. 恋愛・結婚・婚活
*   **ターゲット**: SNS（X）でアクティブな層、婚活市場にいる層。
*   **切り口**: 「婚活を『市場』や『投資』としてロジカルに分析する」。あえて毒舌や強めの意見（炎上要素）を混ぜる。
*   **役割**: 集客のブースター。Xでのバズを起点にnoteプロフィール・他記事へ誘導する。
*   **比較例**: 『ルールズ（古典的恋愛術）』vs『現代のインフルエンサーの恋愛本』

## 2. ハイブリッド運用戦略

「1日1時間・月5万」を達成するため、各ジャンルに役割を持たせる。

| ジャンル | 役割 | 収益化・集客の仕組み |
| :--- | :--- | :--- |
| **投資・経済** | 信頼のベース | 専門性のアピール、高単価マガジン（月額/単発） |
| **ビジネス・自己啓発** | 日常のストック | note内回遊、安定した「スキ」の獲得 |
| **恋愛・婚活** | 集客ブースター | Xでのインプレッション稼ぎ、新規ユーザーの流入 |

## 3. 今後のアクション
1.  **汎用レビュー・エンジンの開発**: ジャンルを問わず「3分核心＋比較」を生成できるAIワークフローの整備。
2.  **トーン＆マナーの統一**: 「すべてを投資・リスク・リターンの観点で捉える」という独自のキャラクター（マリーさん等）の一貫性保持。
3.  **ABテストの実施**: noteでの反応（ビュー、スキ、流入経路）をジャンルごとに比較検証。
