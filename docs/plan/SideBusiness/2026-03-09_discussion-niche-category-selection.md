# 議論メモ: 金融ニッチ記事の有望カテゴリ絞り込み

**日付**: 2026-03-09
**参加**: ユーザー + AI

## 背景・コンテキスト

副業プロジェクト（月5万目標、1日1時間制約）において、note記事のテーマを選定する議論。
既存の調査（Reddit調査、ウェルスマネジメント市場調査、ニッチ戦略調査）を踏まえ、
13テーマ候補から Phase1 で試すテーマを絞り込んだ。

## 議論のサマリー

### 1. 評価軸の決定

5軸（チェックコスト、差別化、需要、継続性、基盤活用）でスコアリングを実施。
ユーザーの判断: **チェックコスト最優先**（1日1時間の制約で毎日更新できるかが鍵）。

### 2. AI系テーマの除外

当初のスコアリングでは「AIで副業してみた」「Claude Codeで作った」が最高スコアだったが、
ユーザーの判断で除外:
- **理由**: レッドオーシャン + すぐ廃れやすい
- **代替方針**: 金融知識系のエバーグリーンコンテンツに注力

### 3. 資産管理助言のレベル設定

「制度活用レベル」を選択:
- NISA × iDeCo の最適組合せ
- ふるさと納税活用
- チェックコスト: 10-15分/記事（公的ソースで確認可能）

### 4. テーマ追加

基本4テーマに加え、以下を追加:
- クレジットカード比較
- ポイント経済圏
- 家計管理/節約術

### 5. カテゴリ再編

投資本レビューを「学習・教養」から「資産形成」カテゴリに移動。

## 決定事項

### 1. AI副業/Claude Code系テーマは除外
- レッドオーシャン + すぐ廃れるため
- 金融知識系エバーグリーンコンテンツに方針転換

### 2. テーマ選定の評価軸はチェックコスト最優先
- 1日1時間の制約下で毎日更新できるかが鍵
- 差別化・需要・継続性・基盤活用は二次的

### 3. 2カテゴリ・7テーマで Phase1 開始

| カテゴリ | テーマ | チェック時間/記事 |
|---|---|---|
| **資産形成** | 制度活用（NISA/iDeCo/ふるさと納税） | 10-15分 |
| | 投資の心理学/基本 | ほぼ0 |
| | 用語解説 | 0 |
| | 投資本レビュー | 5分 |
| **生活のお金** | クレジットカード比較 | 10-15分 |
| | ポイント経済圏 | 5分 |
| | 家計管理/節約術 | 5分 |
