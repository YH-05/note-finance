# X（Twitter）過去ツイート全件取得スキル - 調査結果と判断

## 結論: 実装中止

X ToS の法的リスクを考慮し、このスキルの実装を中止する。

---

## 調査結果サマリー

### 無料でツイートを取得できる手法（2026年2月時点）

| 手法 | 動作状況 | 認証 | ToS違反 |
|------|---------|------|---------|
| **twikit** (d60/twikit) | 良好 | ゲストモードあり | **該当** |
| **twscrape** (vladkens/twscrape) | 良好 | 無料アカウント必要 | **該当** |
| gallery-dl | 条件付き | Cookie必要 | **該当** |
| Playwright ブラウザ自動化 | 可能だが困難 | ログイン推奨 | **該当** |
| snscrape | 不安定（2023年以降） | - | **該当** |
| Nitter | 事実上終了（2024年1月〜） | - | **該当** |
| Wayback Machine | 過去データのみ（2025年5月にブロック） | 不要 | 低リスク |
| X Free API Tier | **読み取り不可**（投稿専用） | 開発者登録要 | 準拠 |
| X Basic API | 読み取り可 | **$200/月** | 準拠 |

### 中止理由: 法的リスク

- **X ToS（2025年5月更新）**: 自動スクレイピングを明示的に禁止。違反時 **$15,000の損害賠償条項**
- 上記の無料手法は**全て** X の内部API を非公式に利用するか、ウェブスクレイピングを行うもので、**ToS 違反に該当**
- X Free API Tier は読み取り（タイムライン・検索）がほぼ不可能で、実質「投稿専用」
- **合法的に無料で読み取りを行う方法は現時点で存在しない**

### 合法的な代替手段（いずれも有料）

| 手法 | コスト | 制限 |
|------|--------|------|
| X Basic API | $200/月 | 10,000 reads/月 |
| X Pro API | $5,000/月 | 1,000,000 reads/月 |
| 商業スクレイピングサービス（Apify等） | $49〜/月 | サービス依存 |
