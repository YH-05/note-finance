# ギャップ分析フレームワーク

## ギャップ種別

### 1. トピックギャップ
競合がカバーしていないトピック、または自分がまだ書いていないトピック。

**評価基準（1-5点）**:
- 5: 市場で誰もカバーしていない重要トピック
- 4: 競合が薄くしかカバーしていない
- 3: 競合がカバーしているが自分はまだ
- 2: 競合が十分カバーしており差別化困難
- 1: 市場で飽和状態

### 2. 深さギャップ
既存の競合記事より深い分析が可能かどうか。

**評価基準（1-5点）**:
- 5: 競合は表面的。データ分析・独自視点で大幅に上回れる
- 4: 競合は一般的。追加データ/分析で差別化可能
- 3: 競合は中程度。部分的に深掘り可能
- 2: 競合は充実。差別化に工夫が必要
- 1: 競合が非常に詳細。上回るのは困難

### 3. 鮮度ギャップ
より新しい情報・データで記事を書けるか。

**評価基準（1-5点）**:
- 5: 競合記事が6ヶ月以上前。最新データで大幅更新可能
- 4: 競合記事が3-6ヶ月前。新しいデータ/イベントあり
- 3: 競合記事が1-3ヶ月前。部分的に更新可能
- 2: 競合記事が1ヶ月以内。差別化は新しい視点が必要
- 1: 競合記事が直近。鮮度では差別化困難

### 4. 読者レベルギャップ
異なるターゲット層向けにコンテンツを提供できるか。

**評価基準（1-5点）**:
- 5: 競合が上級者向けのみ。初心者向け解説の大きな需要あり
- 4: 競合と異なるレベルで書ける余地が大きい
- 3: 部分的に異なるレベルで差別化可能
- 2: 競合が複数レベルをカバー。差別化余地小
- 1: 全レベルで競合が充実

### 5. フォーマットギャップ
異なる形式（データ可視化、比較表、ステップガイド等）で価値を提供できるか。

**評価基準（1-5点）**:
- 5: 競合がテキストのみ。表/チャート/比較で大幅に価値向上可能
- 4: 競合のフォーマットが限定的。異なる形式で差別化可能
- 3: 部分的にフォーマットで差別化可能
- 2: 競合が多様なフォーマット使用。差別化余地小
- 1: フォーマットでの差別化困難

## 機会スコアリング

合計スコア = 5軸の合計（5-25点）

| スコア | 優先度 | アクション |
|--------|--------|-----------|
| 20-25 | 最高 | 即座に記事化を検討 |
| 15-19 | 高 | 次のコンテンツ計画に含める |
| 10-14 | 中 | 時間があれば検討 |
| 5-9 | 低 | 他に機会がなければ検討 |

## 優先度マトリクス

| | 読者関心 高 | 読者関心 低 |
|--|-----------|-----------|
| **ギャップ大** | 最優先（即着手） | 検討（ニッチだが独自性高い） |
| **ギャップ小** | 重要（差別化が鍵） | 低優先（効果が限定的） |
