# beginner 向け X投稿テンプレート

初心者読者（投資経験1-2年以下、NISAを始めたばかり）向けの文体・構成ルール。

---

## 読者プロフィール

- 投資歴: 0〜2年
- 知識水準: NISAを開設したばかり、積立を始めたばかり
- 感情: 「本当にこれで大丈夫？」「難しいこと言われても…」「自分でもできるかな」
- note でよく読む記事: ステップバイステップ解説、「〜してみた」系体験談

## 文体ルール

| 項目 | ルール |
|------|--------|
| 専門用語 | **禁止**（使う場合は括弧内に日本語解説必須） |
| 例) | `DCF（企業の将来価値を計算する方法）` |
| 語尾 | 「〜しました！」「〜です！」（感嘆符OK） |
| 文体 | 柔らかい「です・ます」調 |
| 一人称 | 「私」「筆者」より「あなた」視点 |

## 絵文字ルール

- **積極使用** ✅💡📊🎯📌
- 数字・ポイントの強調に ✅ を使う
- 冒頭の感情フックに 😲 🤔 可
- 1投稿あたり最大 **3個** まで（過多は読みにくい）

## 構成テンプレート

```
{フック（パターンB or C を推奨）}

{核心ポイント 2〜3点}
✅ {ポイント1（できるだけ身近な例で）}
✅ {ポイント2}
✅ {ポイント3（オプション）}

{誘導文（「解説しました！」「詳しくはnoteで」）}
{URL}

{ハッシュタグ 4-5個}
```

## 文字数目安

- **180〜230字**（URLを除く）
- 短く感じても良い。詰め込まない
- 「もっと読みたい」と思わせる余白が重要

## ハッシュタグルール

- **4〜5個**
- 必須: `#投資初心者` `#資産形成`
- カテゴリ別必須タグを追加
  - asset_formation: `#新NISA` `#つみたて投資`
  - asset_management: `#NISA`
  - stock_analysis: `#株初心者`
- 記事キーワードに応じてさらに1-2個追加

## 禁止表現

| 禁止 | 代替 |
|------|------|
| 「ROE」 | 「利益率（ROEという指標）」 |
| 「リバランス」 | 「定期的な比率調整（リバランス）」 |
| 「ボラティリティ」 | 「値動きの激しさ」 |
| 「DCF分析」 | 「企業価値の計算手法」 |
| 「スプレッド」 | 「利回りの差」 |
| 「コンセンサス」 | 「アナリストの予想平均」 |

## 良い例・悪い例

### 良い例（beginner向け）

```
会社員のiDeCo上限、2027年に月1.2万→最大6.2万円へ引き上げ予定✅

✅企業型DC・DBに加入の会社員が対象
✅年収500万円なら月2万円拠出で年間約7.2万円節税
✅10年積み立てると節税効果の累計は約72万円

年収別の節税シミュレーションをnoteで解説しました！
{URL}

#iDeCo #節税 #資産形成 #投資初心者 #老後資金
```

### 悪い例（難しすぎる）

```
iDeCoのDC加入者上限が2027年よりラチェット緩和。
EET課税優遇スキームをフル活用したポートフォリオ最適化の観点から解説。
{URL}
#iDeCo #節税
```
