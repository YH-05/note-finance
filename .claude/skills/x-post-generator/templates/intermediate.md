# intermediate 向け X投稿テンプレート

中級読者（投資経験2〜5年、ニュースを読んで自分で判断できる）向けの文体・構成ルール。

---

## 読者プロフィール

- 投資歴: 2〜5年
- 知識水準: NISA・iDeCoを活用、ETFや個別株も保有、マクロニュースをチェック
- 感情: 「もっと賢く運用したい」「この選択は本当に正しいか確かめたい」「他の人はどうしてる？」
- note でよく読む記事: 比較・検証系、データ分析系、戦略解説

## 文体ルール

| 項目 | ルール |
|------|--------|
| 専門用語 | **適度に使用**。一般的でない場合のみ括弧解説 |
| 例) | 「S&P500」「インデックス」はそのまま使用OK |
| 語尾 | 「〜しました。」「〜です。」（感嘆符は控えめに） |
| 文体 | やや客観的な「です・ます」調 |
| 一人称 | 「私が調べた」「データによると」 |

## 絵文字ルール

- **節度ある使用**（1〜2個）
- 強調・要点整理に絞る
- 過多は「初心者向けっぽく」見えるので注意
- 完全になしでも可

## 構成テンプレート

```
{フック（パターンB or D を推奨）}

{核心ポイント 2〜3点}
・{ポイント1（数字や具体例を交える）}
・{ポイント2}
・{ポイント3（オプション）}

{誘導文（「解説しました。」「詳しくはnoteで。」）}
{URL}

{ハッシュタグ 3-4個}
```

## 文字数目安

- **220〜270字**（URLを除く）
- beginner より少し長め
- 数字・データで密度を上げる

## ハッシュタグルール

- **3〜4個**
- 必須: カテゴリ別デフォルトタグ1-2個
- 記事の核心キーワードを1-2個追加
- 「#投資初心者」は基本的に使わない（読者層とミスマッチ）

## カテゴリ別推奨フック

| category | 推奨フック | 理由 |
|---------|----------|------|
| asset_management | B（疑問形） | 常識への再検証を促す |
| asset_management | B（疑問形） | 戦略の見直しを促す |
| macro_economy | D（宣言型） | 転換点を明確に示す |
| stock_analysis | B（疑問形） | 銘柄選択の判断軸を揺さぶる |
| market_report | D（宣言型） | 週間サマリーを端的に伝える |

## 良い例・悪い例

### 良い例（intermediate向け）

```
「S&P500だけ買えばOK」は本当？

海外の投資コミュニティ（会員200万人超）で調べると：
・S&P500一本は少数派。国際分散や債券組み合わせが主流
・手数料1.3%の差が40年で約2,365万円の差に
・「60:40か70:30か」より「始めて続けること」が大事

Redditの議論から学ぶポートフォリオ配分を解説しました。
{URL}

#資産形成 #インデックス投資 #新NISA #米国株
```
