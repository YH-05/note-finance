# advanced 向け X投稿テンプレート

上級読者（投資経験5年以上、ファンダメンタルズ・マクロ分析を自分で行う）向けの文体・構成ルール。

---

## 読者プロフィール

- 投資歴: 5年以上
- 知識水準: 財務諸表を読める、マクロ政策の含意を自分で考える、クオンツ・定量分析に関心
- 感情: 「自分の見方が正しいか検証したい」「反論があるなら聞いてみたい」「データを見せてくれ」
- note でよく読む記事: 実証分析、逆張り論考、プロの視点

## 文体ルール

| 項目 | ルール |
|------|--------|
| 専門用語 | **積極使用**。解説は省略 |
| 語尾 | 「〜を検証しました。」「〜が示しています。」（断定調） |
| 文体 | 簡潔・事実ベース。感情的な表現は排除 |
| 一人称 | 「データによると」「過去X回の検証では」 |

## 絵文字ルール

- **原則なし**
- 使う場合は表現上絶対に必要な場合のみ（1個まで）

## 構成テンプレート

```
{フック（パターンA or D を推奨）}

{核心ポイント 2〜3点（データ・数字・指標を含む）}
・{ポイント1（具体的な数値を明示）}
・{ポイント2（因果関係やメカニズムを1文で）}
・{ポイント3（含意・投資判断への接続）}

{誘導文（「〜を検証しました。」「詳細はnoteで。」）}
{URL}

{ハッシュタグ 3個}
```

## 文字数目安

- **230〜280字**（URLを除く）
- 情報密度を高く保つ
- 「ふわっとした」表現は削除

## ハッシュタグルール

- **3個**（必要最小限）
- カテゴリの核心キーワードで構成
- 「#投資初心者」「#NISAで増やす」など初心者向けタグは使わない

## カテゴリ別推奨フック

| category | 推奨フック | 理由 |
|---------|----------|------|
| asset_management | B（疑問形） | ポートフォリオ理論の検証を促す |
| asset_management | A（逆説） | 常識への反論・心理バイアスの深掘り |
| macro_economy | A（逆説） | コンセンサスへの反論 |
| stock_analysis | D（宣言） | バリュエーション判断の表明 |
| quant_analysis | D（宣言） | 定量分析結果の端的な提示 |

## 良い例・悪い例

### 良い例（advanced向け）

```
今の高配当ブームは3年以内に崩壊する。

・10年債利回りが4%超で推移する局面、高配当株のバリュエーションは過去4サイクルで平均-23%
・FCFイールド3%以下の「高配当銘柄」が全体の38%——持続可能性に疑問
・グロース金利感応度が高まる金融引き締め後期では、配当利回りだけで保有する根拠は薄い

高配当株サイクルの終点を識別する指標3つを検証しました。
{URL}

#高配当株 #バリュー投資 #米国株
```

### 悪い例（初心者すぎる）

```
高配当株っていいですよね！💡
初心者でも分かりやすく解説しました！
✅インカムゲインが安定
✅ NISA口座で節税も
{URL}
#投資初心者 #高配当株 #NISA #資産形成
```
