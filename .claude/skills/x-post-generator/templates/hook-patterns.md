# フックパターン対応表 — 4パターン × 3層

X投稿の書き出しパターンと層別の使い分けガイド。

---

## パターン定義

| パターン | 名称 | 核心 | 最適コンテキスト |
|---------|------|------|----------------|
| A | 逆説型 | 「〜ではない、〜だから」 | 感情・心理系、行動バイアス、思い込み打破 |
| B | 疑問形型 | 「〜は本当？」 | 常識検証系、比較・選択系、ランキング系 |
| C | 数字型 | 「{数字}。{背景}。」 | 制度変更、シミュレーション、統計、ファクト系 |
| D | 宣言型 | 「{大胆な主張}。{理由}。」 | マクロ転換点、逆張り分析、上級者向け見解 |

---

## 12パターン対応表

### パターンA × 3層

**A × beginner（逆説型・初心者向け）**
- 文型: `{通説}は「{よくある原因}」のせいではありません。実は{本当の理由}なんです。`
- 感情軸: 驚き → 「自分だけじゃなかった」という安心
- 絵文字: 冒頭に😲や❗可
- 例文:
  ```
  投資が怖いのは「知識不足」のせいではありません。

  脳が「損をする可能性」に過剰反応するように設計されているからです。
  ```

**A × intermediate（逆説型・中級者向け）**
- 文型: `{専門的な通説}は{誤解}ではありません。{本質的な理由}です。`
- 感情軸: 「損してた」「見落としていた」という気づき
- 絵文字: あっても1個まで
- 例文:
  ```
  暴落で売りたくなるのは「意志が弱いから」ではありません。

  損失の痛みは利益の喜びの約2倍——プロスペクト理論が証明しています。
  ```

**A × advanced（逆説型・上級者向け）**
- 文型: `{通説}は{一般論}とは限りません。{データ・根拠}。`
- 感情軸: 反論・知的刺激・「自分の見方が正しかった」
- 絵文字: 原則なし
- 例文:
  ```
  「分散投資でリスクは下がる」は常に成立しません。

  相関係数が1に近づく局面（2022年債券株式同時安）では分散効果がゼロになる。
  ```

### パターンB × 3層

**B × beginner（疑問形型・初心者向け）**
- 文型: `「{初心者がよく思い込んでいること}」って本当？💡`
- 感情軸: 「自分も思ってた！」という共感 → 答えを知りたい好奇心
- 絵文字: 疑問符 + 1-2個の絵文字OK
- 例文:
  ```
  「オルカン買えば一生安泰？」って本当？💡

  実は、年齢によって最適なファンドは変わります。
  ```

**B × intermediate（疑問形型・中級者向け）**
- 文型: `「{投資の常識的な問い}」は本当？`
- 感情軸: 知的好奇心・「実はどうなんだろう」
- 絵文字: あっても1個
- 例文:
  ```
  「S&P500だけ買えばOK」は本当？

  海外投資コミュニティ（200万人超）の実態を調べると、意外な結論が出ました。
  ```

**B × advanced（疑問形型・上級者向け）**
- 文型: `「{専門的な仮説・通説}」は本当に成立するか。`
- 感情軸: 反論・検証欲求・プロ同士の議論を喚起
- 絵文字: なし
- 例文:
  ```
  「高配当株は割高になりにくい」は本当に成立するか。

  FCFイールドとペイアウトレシオのデータで検証しました。
  ```

### パターンC × 3層

**C × beginner（数字型・初心者向け）**
- 文型: `{身近な制度/名詞}、{変化や効果の数字}✅`
- 感情軸: 「へぇ！知らなかった」という発見 → お得感・行動意欲
- 絵文字: 数字強調に✅や📊
- 例文:
  ```
  iDeCo上限、2027年に月1.2万→最大6.2万円へ引き上げ予定✅

  年収500万円なら年間約7.2万円の節税効果。
  ```

**C × intermediate（数字型・中級者向け）**
- 文型: `{制度/現象}、{具体的な数字と変化}。`
- 感情軸: 「この数字は見逃せない」という実務的緊迫感
- 絵文字: あっても1個
- 例文:
  ```
  原油が150ドルを超えた場合、日本の貿易収支は年間約8兆円悪化する試算。

  物価・企業収益への影響シナリオを整理しました。
  ```

**C × advanced（数字型・上級者向け）**
- 文型: `{指標/データ}が{数値}を示した。{含意}。`
- 感情軸: データに基づく判断の確信
- 絵文字: なし
- 例文:
  ```
  Q2のFCFイールド中央値が4.2%に到達。

  過去データと照らすと、このレンジは3回中2回で翌年のアウトパフォームと相関する。
  ```

### パターンD × 3層

**D × beginner（宣言型・初心者向け）**
- 文型: `{大事なことをひとことで断言}。{その理由を身近な言葉で補足}。`
- 感情軸: 「そうか、そういうことか」という納得感・行動喚起
- 絵文字: 🎯や📌などシンボリックなもの1個可
- 例文:
  ```
  📌 長期投資で大事なのは「銘柄選び」より「やめないこと」。

  10年間積み立てた人の9割以上がプラスになっています。
  ```

**D × intermediate（宣言型・中級者向け）**
- 文型: `{転換点・判断を断言}。{根拠となる指標・事実}。`
- 感情軸: 「この判断は正しい」という確信・行動意欲
- 絵文字: あっても1個
- 例文:
  ```
  日銀が0.5%を超えた。住宅ローン戦略の転換点です。

  変動型のリスクが急上昇した3つの根拠を整理しました。
  ```

**D × advanced（宣言型・上級者向け）**
- 文型: `{逆張り的な主張を端的に断言}。{データ・ロジックで補強}。`
- 感情軸: 知的刺激・反論したくなる緊張感・「自分も考えてみよう」
- 絵文字: なし
- 例文:
  ```
  今の高配当ブームは3年以内に崩壊する。

  金利上昇局面では高配当株のバリュエーション是正が先行する——過去4サイクルのデータが示している。
  ```
