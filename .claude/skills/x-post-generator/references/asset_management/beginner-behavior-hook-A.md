# 本文

暴落で売りたくなるのは「意志が弱いから」ではありません。

脳がそう動くように設計されているからです。

・損失の痛みは利益の喜びの約2倍（プロスペクト理論）
・「買値に戻るまで待つ」はアンカリングの罠
・SNSの含み益自慢はFOMOを生む生存者バイアス

積立初期は「市場の値動き」より「継続すること」が重要——NISAで知っておくべき脳の裏切りを解説しました。
