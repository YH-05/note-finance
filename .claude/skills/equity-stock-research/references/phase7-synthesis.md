# Phase 7: Investment Synthesis — 詳細質問リスト

## 統合的質問（全ソース横断で必ず問うべき5つの質問）

### 1. Bull Case
対象企業に最も強気になれる要因を3つ挙げてください。各ソースからの根拠も含めて。

**深掘りの視点:**
- その強気要因は既にコンセンサスに織り込まれているか？
- Bull caseが実現した場合のアップサイドは何%か？
- Bull caseの実現確率をどう評価するか？

### 2. Bear Case
対象企業に最も慎重になるべき要因を3つ挙げてください。定量的なダウンサイドシナリオを含めて。

**深掘りの視点:**
- Bear caseが実現した場合のダウンサイドは何%か？
- 損失を限定するためのストップロス水準は？
- Bear caseの早期警戒シグナルは何か？

### 3. Consensus vs Reality
セルサイドのコンセンサス見通しと、会社側の発言・実績データに乖離がある点はどこですか？

**深掘りの視点:**
- その乖離は市場に認識されているか？
- 乖離が解消される場合、株価にどう影響するか？
- 会社の実績データはコンセンサスの上振れ/下振れどちらを示唆しているか？

### 4. What's Different Now
直近1-2年でファンダメンタルズが最も大きく変わった点は何ですか？

**深掘りの視点:**
- その変化は一時的か構造的か？
- バリュエーションはその変化を適切に反映しているか？
- 競合も同様の変化を経験しているか、それとも対象企業固有か？

### 5. Under-discussed
複数のレポートで十分に議論されていないが、投資判断に重要と思われる論点はありますか？

**深掘りの視点:**
- なぜ議論されていないのか？（データ不足？コンセンサスの盲点？）
- その論点が顕在化した場合のインパクトは？
- 自社のリサーチでエッジを持てる領域か？

## バイサイド固有の統合フレームワーク

### 投資判断の最終チェックリスト

| 観点 | 確認事項 |
|------|---------|
| ポジショニング | ポートフォリオ内での役割（コア/サテライト/イベントドリブン） |
| サイジング | リスクに見合ったポジションサイズか |
| タイミング | カタリストの時間軸は投資期間と整合するか |
| 出口戦略 | どの条件で利確/損切りするか |
| 相関 | 既存ポートフォリオとの相関は？リスク分散に寄与するか |

### 「裏を読む」チェックポイント

- **経営陣のトーン変化**: 前回決算と比べて微妙にトーンが変わった点はないか
- **IR資料の変更**: スライドの構成変更、KPIの表示方法変更は何かのシグナル
- **非開示の情報**: 以前は開示していたが今回開示しなくなった情報はないか
- **内部者取引**: インサイダーの売買動向は経営陣の自信/懸念を反映する
- **株主構成の変化**: 大口投資家の参入/撤退は注目すべきシグナル
