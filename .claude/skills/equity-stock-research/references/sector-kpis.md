# セクター別 KPI カタログ

Phase 3 の「セクター固有KPI動的深掘り」で参照するカタログ。
対象企業のセクターを特定した後、該当セクションのKPIリストを使って追加の質問を生成する。

> このカタログに該当セクターがない場合は、Phase 1-2 の情報を基に「動的質問生成」プロセス（SKILL.md参照）で KPI を推論する。

---

## Telecom（通信）

| KPI | 説明 | 確認ポイント |
|-----|------|------------|
| ARPU (Blended/Prepaid/Postpaid) | 加入者1人あたり月間収益 | 四半期推移、Prepaid→Postpaid移行の影響 |
| 加入者数 | モバイル加入者数 | アクティブの定義、SIM vs ユニークユーザー |
| チャーン率 | 月間解約率 | 業界平均との比較、Prepaid/Postpaid別 |
| Data Traffic (PB) | 総データ通信量 | 1加入者あたりデータ使用量の推移 |
| Data Yield (IDR/GB) | 1GBあたりデータ収益 | 低下トレンドの速度とARPUとのバランス |
| 基地局数 (BTS) | 4G/5G内訳 | カバレッジ率、競合との比較 |
| Capex/Revenue | 設備投資対売上比率 | 投資サイクルの位置 |
| 周波数帯域幅 (MHz) | 保有スペクトラム | オークション予定、競合比較 |

### 通信セクター固有の質問テンプレート
- ARPU（Blended/Prepaid/Postpaid）の四半期推移と変動要因は？
- Data trafficの推移と1加入者あたりデータ使用量は？
- Data yield (IDR/GB)のトレンドと低下要因は？
- 基地局数とカバレッジの推移は？4G/5Gの内訳も。
- タワー保有 vs リースの戦略方針は？
- ファイバー敷設距離とFTTH展開計画は？
- ネットワーク品質の第三者評価（Opensignal、Ookla等）の結果は？

## カタログにないセクターの場合

以下のフレームワークでKPIを推論する:

1. **収益構造を分解**: 売上 = 何 x 何？（量 x 単価、顧客数 x ARPU、面積 x 賃料、等）
2. **効率指標を特定**: その業界で「うまく経営できているか」を測る指標は？（稼働率、歩留まり、回転率等）
3. **先行指標を探す**: 将来の売上を予測できる指標は？（受注残、パイプライン、予約数等）
4. **業界レポートを参照**: IR資料やセルサイドレポートで頻出するKPIは経営陣自身が重要と認識している指標
5. **競合との比較軸**: 同業他社との差を最も端的に表す指標は？
