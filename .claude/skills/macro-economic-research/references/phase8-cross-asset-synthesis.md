# Phase 8: Cross-Asset Synthesis — 詳細質問リスト

## 統合的質問（全Phase横断で必ず問うべき5つの質問）

### 1. Regime Call
現在のマクロレジームは何か（Goldilocks/Reflation/Stagflation/Deflation）？今後6-12ヶ月でレジームが変化する可能性はどの程度か？

**深掘りの視点:**
- レジーム判定に最も確信が持てるデータポイントは？
- レジーム転換を最初に示唆するであろう先行指標は？
- 現在のレジーム判定にバイアスがかかっていないか？（確証バイアスの自己チェック）

### 2. Consensus vs Our View
市場コンセンサスと自分のマクロビューが最も乖離するポイントはどこか？

**深掘りの視点:**
- その乖離がポジションの機会につながるか？
- 自分の見方が正しい場合の市場インパクトの大きさは？
- 自分の見方が間違っている場合のリスクは？（非対称性の評価）

### 3. Key Catalyst Timeline
今後3ヶ月の重要イベントカレンダーと、各イベントの想定インパクトは？

**深掘りの視点:**
- 最もマーケットムービングなイベントはどれか？
- サプライズが起きた場合のシナリオは？
- イベント前のポジショニング調整のタイミングは？

### 4. Tail Risks
確率は低いが実現時のインパクトが大きいシナリオ（左尾/右尾の両方）は？

**深掘りの視点:**
- 左尾（ネガティブ）: リセッション急加速、金融危機、地政学エスカレーション
- 右尾（ポジティブ）: 生産性革命、予想外の平和合意、供給ショックの解消
- テールリスクのヘッジコストは合理的か？

### 5. Asset Allocation Implications
現在のマクロレジームと見通しに基づく、資産クラス別の方向性は？

**深掘りの視点:**
- 株式（先進国/EM、セクター配分）
- 債券（デュレーション、クレジット、IG vs HY）
- FX（ドルのロング/ショート、EM通貨のセレクション）
- コモディティ（エネルギー、金属、農産物）
- オルタナティブ（ヘッジファンド、PE、実物資産）

## クロスアセットのシグナル整合性チェック

異なる資産クラスが発するシグナルの整合性を確認する:

| シグナルの組み合わせ | 整合性チェック |
|-------------------|-------------|
| 株高 + 金利上昇 | 整合的: 景気回復/リフレーション |
| 株高 + 金利低下 | 要注意: 流動性相場？質への逃避の予兆？ |
| 株安 + 金利低下 | 整合的: リスクオフ/景気後退懸念 |
| 株安 + 金利上昇 | 要注意: スタグフレーション？財政リスク？ |
| ドル高 + コモ安 | 整合的: グローバル需要減退 |
| ドル高 + コモ高 | 要注意: 供給ショック？地政学リスク？ |
| VIX上昇 + スプレッドワイド | 整合的: ストレス環境 |
| VIX低位 + スプレッドタイト | 整合的だが要注意: コンプレイシェンシーリスク |

## equity-stock-research への橋渡し

マクロ分析の結果を個別銘柄分析に接続するフレームワーク:

| マクロ結論 | 個別銘柄への影響 | equity-stock-research のPhase |
|-----------|---------------|---------------------------|
| 金利上昇見通し | デュレーションの長い資産（REIT、ユーティリティ）にネガティブ | Phase 5: バリュエーション前提の見直し |
| ドル高見通し | EM収益比率の高い企業にネガティブ | Phase 4: 為替感応度の確認 |
| 原油高見通し | エネルギーセクターにポジティブ、航空・物流にネガティブ | Phase 2: 業界環境、Phase 6: コスト感応度 |
| リセッション確率上昇 | ディフェンシブへの回転 | Phase 7: Bear case の再評価 |
| 中国景気回復 | コモディティ、EM、資源関連にポジティブ | Phase 2: 市場環境 |

### 「裏を読む」チェックポイント

- **ポジショニングの偏り**: ファンドマネージャー調査（BofA FMS等）で最も混雑したトレードは何か？混雑トレードの巻き戻しリスクは？
- **バリュエーションの自己強化**: 株高→資産効果→消費増→GDP上昇→更なる株高のフィードバックループは持続可能か？
- **ボラティリティの構造**: 低ボラティリティ環境の持続はレバレッジの拡大を誘い、将来のショック時のダウンサイドを増幅する
- **中銀プット**: 市場が「中銀が下支えする」と信じている度合い。この信念が揺らぐイベントは何か？
