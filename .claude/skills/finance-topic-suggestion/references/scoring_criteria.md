# 金融記事トピックのスコアリング基準

提案する各トピックについて、以下の5つの基準でスコアリングを行い、合計スコア（最大50点）を算出します。

## 1. 時事性 (Timeliness)
- **8-10点**: 今週の重要なイベント、最新のニュース、リアルタイムのトレンド。
- **5-7点**: 今月、四半期ベースの注目のイベント、周期的なテーマ。
- **1-4点**: 特筆すべき時期の定めがないエバーグリーンな内容。

## 2. 情報の入手性 (Information Availability)
- **8-10点**: 公的データ、詳細なレポート、豊富なニュースソースがある。
- **5-7点**: 十分なソースがある。
- **1-4点**: ソースが乏しい、独自調査が不可欠。

## 3. 読者関心度 (Reader Interest)
- **8-10点**: SNSでの議論が活発、検索トレンドが急上昇中、主要指数の動きに関連。
- **5-7点**: 一定の層には関心が強い。
- **1-4点**: マニアックなテーマ、ニッチなトピック。

## 4. 執筆の実現可能性 (Feasibility)
- **8-10点**: 3000-5000字程度で完結し、構造化が容易。
- **5-7点**: 追加の調査が必要だが、構成は可能。
- **1-4点**: 大規模な調査や、高度な専門性が不可欠。

## 5. 独自性 (Uniqueness)
- **8-10点**: 既存の記事にない視点、自分なりの見解を付加可能。
- **5-7点**: 標準的な解説だが、最新情報を反映。
- **1-4点**: ネット上の既存記事と重複、類似記事が多数。
