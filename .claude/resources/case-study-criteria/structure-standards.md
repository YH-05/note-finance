# 構成バランス 評価基準

## 概要

事例分析型記事がテンプレート（A/B/C）に準拠し、セクション間の文字量バランスが適切かを評価する。

## 共通基準

### 総文字量

| 項目 | 基準 |
|------|------|
| 目標文字量 | 6,000-8,000字（データカード除く） |
| 最低文字量 | 5,500字（これ以下はリジェクト） |
| 上限文字量 | 9,000字（これ以上は分割を検討） |

### [EMBED] マーカー

- テンプレートで指定された位置に [EMBED] マーカーがあるか
- 各マーカーの前後に分析文脈が書かれているか
- マーカー数がテンプレート指定と一致しているか

### テンポ

- 長文（100字超）が3段落以上連続しないこと
- 短文インパクト（20字以下の文）が適度に使われていること
- 箇条書き・表の活用で視覚的に読みやすいこと

## テンプレート別セクション基準

### テンプレートA: ジャンル別事例分析

| セクション | 目安文字量 | 必須 |
|-----------|-----------|------|
| 導入 | 600-1,000字 | YES |
| 事例紹介 | 2,000-2,800字 | YES |
| パターン分析 | 1,500-2,000字 | YES |
| 落とし穴 | 800-1,200字 | YES |
| ロードマップ | 800-1,200字 | YES |
| まとめ | 400-700字 | YES |

[EMBED] 期待位置: 事例紹介、落とし穴、ロードマップ

### テンプレートB: ジャンル横断共通点分析

| セクション | 目安文字量 | 必須 |
|-----------|-----------|------|
| 導入 | 600-1,000字 | YES |
| 共通点1 | 1,000-1,400字 | YES |
| 共通点2 | 1,000-1,400字 | YES |
| 共通点3 | 1,000-1,400字 | YES |
| 共通点4-5 | 各800-1,000字 | NO |
| 意外だった点 | 800-1,200字 | YES |
| まとめ | 500-800字 | YES |

[EMBED] 期待位置: 共通点1、共通点3

### テンプレートC: AI一人スタートアップ事例分析

| セクション | 目安文字量 | 必須 |
|-----------|-----------|------|
| 導入 | 600-1,000字 | YES |
| 事例深掘り（1-3件） | 各1,200-2,000字（合計2,500-4,000字） | YES |
| パターン分析 | 1,200-1,800字 | YES |
| 日本市場への示唆 | 800-1,200字 | YES |
| まとめ | 400-700字 | YES |

[EMBED] 期待位置: 事例深掘り、日本市場への示唆

## セクション偏差の評価

| 偏差率 | 評価 |
|--------|------|
| ±10%以内 | 適正（減点なし） |
| ±10-25% | 軽微な偏り（low） |
| ±25-50% | 偏りあり（medium） |
| ±50%超 | 大きな偏り（high） |
