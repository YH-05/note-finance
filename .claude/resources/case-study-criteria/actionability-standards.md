# 実用性 評価基準

## 概要

事例分析型記事が「すごい事例の紹介」で終わらず、読者が自分の行動に活かせる内容になっているかを評価する。
note.com のターゲット読者（会社員、副業希望者、投資初心者）の視点で判断する。

## カテゴリ別基準

### 1. 再現性評価（reproducibility）

#### チェック項目

- 成功の「再現可能性」について言及があるか
- 生存者バイアスへの言及があるか（成功事例だけでなく、同じ手法で失敗した人の存在）
- 成功の条件（スキル・タイミング・市場環境等）が明示されているか
- 「この成功は例外的か、再現可能か」の判断が示されているか

#### スコアリング

| 状態 | スコア |
|------|--------|
| 再現性を条件付きで明確に評価 | 90-100 |
| 再現性への言及があり、条件も一部示されている | 70-89 |
| 再現性への言及が曖昧 | 50-69 |
| 再現性の評価がない | 0-49 |

### 2. 条件明示（conditions）

#### チェック項目

- 必要なスキル・知識が明示されているか
- 必要な時間投資（週何時間、何ヶ月等）が具体的か
- 必要な資金（初期投資、ランニングコスト）が明示されているか
- 成功までの想定期間が現実的に示されているか
- 読者の属性別（エンジニア vs 非エンジニア等）の条件分岐があるか

### 3. 行動ステップ（steps）

#### チェック項目

- 具体的なロードマップ or ステップが示されているか
- 各ステップが実行可能な粒度か（「マーケティングをする」ではなく「X で #タグ を使って週3回発信」）
- マイルストーン（1ヶ月目、3ヶ月目等）が示されているか
- 各ステップでの「つまずきやすいポイント」が示されているか

### 4. 読者接地（grounding）

#### チェック項目

- 「会社員で週10時間しかない人にとっては…」等の読者の状況への引き寄せがあるか
- セクション末尾に問いかけや共感ポイントがあるか
- 「自分にも関係あるか？」に応える記述があるか
- 読者の不安・疑問を先回りして解消しているか

#### 読者ペルソナ

事例分析型の主要ターゲット:

| ペルソナ | 特徴 |
|---------|------|
| 副業未経験の会社員 | 週10-15時間、初期投資は月1万まで、リスク回避型 |
| 副業経験あり（月1-3万） | スキルアップしたい、月5万超えが目標 |
| 起業に興味がある人 | 技術力あり、リスクテイクに前向き、海外事例に関心 |
| 投資初心者 | つみたてNISA検討中、損失が怖い |

## 危険シグナル

以下のいずれかに該当する場合、実用性スコアを大幅に減点:

| シグナル | 減点 |
|---------|------|
| 成功事例のみで失敗事例なし | -15 |
| 読者の状況への引き寄せが一切ない | -15 |
| 「すごい」「天才的」等の感嘆だけで分析なし | -20 |
| 条件・コスト・時間の記述がゼロ | -10 |
