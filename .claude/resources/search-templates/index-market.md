# 株価指数・市場全体 検索クエリテンプレート

## 市場概況

主要指数の週次・月次パフォーマンスや市場全体のトレンドを把握する。

| クエリ | ユースケース |
|--------|-------------|
| `"S&P 500 weekly performance {YYYY}"` | S&P500の週次パフォーマンス確認 |
| `"S&P 500 {PERIOD} analysis outlook"` | 特定期間の分析・見通し |
| `"stock market rally selloff {PERIOD}"` | 上昇・下落の背景要因調査 |
| `"NASDAQ composite {PERIOD} analysis"` | ナスダック市場の動向 |
| `"Dow Jones industrial average {PERIOD}"` | ダウ平均の動向 |
| `"US stock market weekly recap {PERIOD}"` | 週次市場まとめ |
| `"stock market all-time high record {YYYY}"` | 最高値更新時の背景 |
| `"market volatility VIX {PERIOD}"` | ボラティリティ・VIX動向 |
