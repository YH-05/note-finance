# 競合コンテンツ調査 検索クエリテンプレート

## note.com検索

note.com上の金融コンテンツを調査し、トレンドやギャップを発見する。

| クエリ | ユースケース |
|--------|-------------|
| `"site:note.com 米国株 {PERIOD}"` | note上の米国株コンテンツ |
| `"site:note.com 投資 初心者 {KEYWORD_JA}"` | 初心者向け投資コンテンツ |
| `"site:note.com マーケットレポート {PERIOD}"` | マーケットレポート系記事 |
| `"site:note.com 資産形成 {KEYWORD_JA}"` | 資産形成コンテンツ |
| `"site:note.com 高配当 株 {PERIOD}"` | 高配当株コンテンツ |
| `"site:note.com S&P500 インデックス"` | S&P500・インデックス投資 |
| `"site:note.com NISA 投資信託 {PERIOD}"` | NISA・投信関連 |
| `"site:note.com 決算 分析 {PERIOD}"` | 決算分析記事 |

## トレンド把握

| クエリ | ユースケース |
|--------|-------------|
| `"金融 ブログ 人気 {PERIOD}"` | 人気の金融ブログ・記事 |
| `"投資 コンテンツ トレンド {YYYY}"` | 投資コンテンツのトレンド |
| `"note 金融 ランキング {PERIOD}"` | noteの金融カテゴリ人気 |
| `"個人投資家 情報収集 方法 {YYYY}"` | 個人投資家の情報収集手段 |
| `"投資 SNS インフルエンサー {PERIOD}"` | 投資系インフルエンサーの動向 |
