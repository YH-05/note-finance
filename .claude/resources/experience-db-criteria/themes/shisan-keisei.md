## リアリティ固有チェック

### 資産形成特有の具体性
- [ ] 金融商品名が具体的か（eMAXIS Slim 全世界株式、S&P500等）
- [ ] 証券口座のプラットフォーム名があるか（SBI証券、楽天証券等）
- [ ] 制度名が正確か（つみたてNISA、新NISA、iDeCo、企業型DC等）
- [ ] 資産推移の数字がスペックカードと整合しているか
- [ ] 利回り・リターンの数字が現実的か（年利20%超は要注意）
- [ ] 月々の投資額が手取り・生活費から逆算して妥当か

### 資産形成特有のNGパターン
- 特定銘柄の推奨（投資助言に該当するリスク）
- 「必ず儲かる」「損しない」等の断定表現
- リスクへの言及なしに投資を推奨
- 過度に楽観的なリターン想定
- 金融庁の注意喚起に該当する表現

### 金融コンプライアンス注意点
- [ ] 「投資は自己責任」の注記があるか（必須ではないが推奨）
- [ ] 「過去のリターンは将来を保証しない」のニュアンスが含まれているか
- [ ] 特定商品の購入を直接推奨していないか
- [ ] 体験談として語り、投資助言にならないラインを維持しているか

## 共感度固有チェック

### 資産形成特有の共感ポイント
- [ ] 「投資＝ギャンブル」の先入観とその払拭過程が描かれているか
- [ ] 「親世代の助言 vs 正しい知識」の葛藤があるか
- [ ] 「銀行預金の利息37円」的な具体的ショックがあるか
- [ ] 「アプリの評価額を何度も見てしまう」心理が描かれているか
- [ ] 「知らなかった機会損失」の計算とその感情があるか
- [ ] 暴落時の不安と「売らなかった理由」が具体的か

### 読者が離脱しやすいポイント
- 金融用語の説明なしの連発
- 「私はこんなに増えました」の資産自慢
- 投資手法の解説に偏りすぎ（ハウツー記事化）
- リスクへの言及なし（楽観的すぎて信頼性を失う）

## 埋め込みリンク固有チェック

### 適切なリソース例
| セクション | リソース種類 | 例 |
|-----------|-------------|-----|
| きっかけ | YouTube | つみたてNISA/新NISA入門解説（初心者向け） |
| 最初の試練 | note / ブログ | 暴落時のメンタル管理、長期投資の心構え |

### 不適切なリソース例
- 特定証券口座の口座開設アフィリエイトリンク
- 個別株の売買推奨記事
- FX・暗号資産等のハイリスク商品の紹介
- 「10倍株を見つける方法」系コンテンツ

## 文字量バランス固有チェック

### 重要度の高いセクション
- **最初の試練**（1,000-1,500字）: 暴落体験の心理描写が核心
- **転機**（1,000-1,500字）: 「お金の使い方の変化」の丁寧な描写
- **現在地**（1,000-1,500字）: 数字テーブル + 内面の成長の両方が必要

### 数字の扱い
- 資産推移は表形式推奨
- 数字の前後に必ず「その数字が生活にどう影響したか」を添える
- 複利計算等は入れすぎない（ハウツー化を避ける）
