# 文字量バランス基準（共通）

## 概要

記事の総文字量、セクション間の配分、情報密度、読みのテンポを評価する基準。
目標は「電車内10-15分で読了」のペースを維持すること。

## 基本数値

| 項目 | 基準 |
|------|------|
| 目標文字量 | 6,000-8,000字（スペックカード・合成ソースメモ除く） |
| 最低文字量 | 5,500字（これ以下はリジェクト） |
| 最大文字量 | 9,000字（これ以上は冗長） |
| 読了時間の目安 | 10-15分 |

## 評価項目

### 1. 総文字量の適正性（total_wordcount）

記事全体の文字量が目標範囲に収まっているか。

| 状態 | スコア |
|------|--------|
| 6,000-8,000字 | 100 |
| 5,500-5,999字 or 8,001-9,000字 | 70 |
| 5,000-5,499字 or 9,001-10,000字 | 40 |
| 5,000字未満 or 10,000字超 | 0（リジェクト） |

**チェック項目**:
- [ ] スペックカード・合成ソースメモを除いた本文文字量を計測
- [ ] 目標範囲内か
- [ ] リジェクトラインを下回っていないか

### 2. セクション間のバランス（section_distribution）

各セクションがテンプレートの文字量ガイドラインに沿っているか。

**許容範囲**: ガイドライン値の ±30%

**婚活テンプレートの例**:
| セクション | ガイドライン | 許容範囲 |
|-----------|-------------|---------|
| きっかけ | 800-1,200字 | 560-1,560字 |
| 迷走期 | 1,200-1,800字 | 840-2,340字 |
| 転機 | 1,000-1,500字 | 700-1,950字 |
| 出会い | 1,000-1,500字 | 700-1,950字 |
| 決断 | 600-1,000字 | 420-1,300字 |
| 振り返り | 800-1,200字 | 560-1,560字 |

**チェック項目**:
- [ ] 各セクションの文字量を計測
- [ ] テンプレートガイドラインの ±30% 以内か
- [ ] 極端に短い / 長いセクションがないか
- [ ] 最長セクションと最短セクションの比率が 4:1 以内か

### 3. 情報密度の均一性（density）

セクションごとの情報密度（エピソード数、数字の頻度等）に偏りがないか。

**密度指標**:
- エピソード数 / セクション
- 数値データ数 / セクション
- 具体的固有名詞数 / セクション
- 読者への問いかけ数 / セクション

**チェック項目**:
- [ ] エピソードが特定セクションに集中していないか
- [ ] 数値が出てくるセクションと出てこないセクションの差が大きすぎないか
- [ ] 「スカスカ」に感じるセクションがないか
- [ ] 情報過多で読みにくいセクションがないか

### 4. 読みのテンポ・緩急（pacing）

段落の長さ、文の長さ、改行の頻度が適切で、テンポよく読めるか。

**note.com のモバイル読者を想定**:
- 段落の長さ: スマホで3-5行（80-160字）が目安
- 文の長さ: 30-50字（1文が80字を超えると読みにくい）
- 改行・区切り: 300-500字ごとに視覚的な区切り

**テンポの緩急パターン**:
```
長めの描写段落（150字）
  ↓
短い一文段落（20字）← インパクト
  ↓
中くらいの段落（100字）
  ↓
会話や数字の挿入 ← テンポチェンジ
```

**良い例**:
```
27歳で貯金がほぼゼロ。          ← 短い一文（インパクト）

同期の田中は去年車を買った。    ← 中くらいの段落
インスタにBMWの写真を...

27歳で、飲み会1回のリカバリー   ← 短い一文（反復でリズム）
に1週間かかる生活。
```

**チェック項目**:
- [ ] 80字以上の文が全体の20%以下か
- [ ] 短い一文段落がインパクトのある場面で使われているか
- [ ] 5行以上の段落が連続していないか
- [ ] セクション間に適切な区切り（---）があるか
