# データ正確性基準

## 許容誤差テーブル

| データタイプ | 許容誤差 | 備考 |
|-------------|---------|------|
| 株価 | ±0.01 | 小数点以下2桁 |
| 変動率 | ±0.1% | パーセンテージ |
| 大きな数値（売上等） | ±1% | 相対誤差 |
| 経済指標 | ±0.1 | 絶対誤差 |
| バリュエーション（P/E等） | ±0.1 | 小数点以下1桁 |
| 配当利回り | ±0.01% | パーセンテージ |

## 検証対象データタイプ

### 株価データ
- 終値、始値、高値、安値
- 出来高
- 52週高値/安値

### 変動率
- 日次/週次/月次/年次リターン
- 前期比/前年比

### 決算データ
- 売上高、営業利益、純利益
- EPS、配当

### バリュエーション
- P/E, P/B, PEG
- 配当利回り、時価総額

### 経済指標
- GDP、CPI、失業率、金利

## データタイプ別検証方法

### 株価検証
1. market_data/data.json から該当シンボルを検索
2. 該当日のデータを取得
3. 記事の値と比較
4. 許容誤差内か判定

### 変動率検証
1. 元データから変動率を再計算: (終値 - 基準値) / 基準値 x 100
2. 記事の値と比較
3. 許容誤差内か判定

### 経済指標検証
1. market_data/data.json (economic セクション) から検索
2. 記事の値と比較
