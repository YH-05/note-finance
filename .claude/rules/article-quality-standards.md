# 記事品質ルール（note.com 向け）

記事執筆・批評・修正に関わる全エージェント・スキル・コマンドが遵守すべきルール。

## ルール1: マークダウン表は必ず画像化する

note.com ではマークダウン表が正しくレンダリングされないため、**記事内の全てのマークダウン表を `/generate-table-image` スキルでPNG画像に変換**し、画像参照で埋め込むこと。

### 手順

1. 表データのJSONを作成（`.tmp/table_*.json`）
2. `uv run --with playwright python scripts/generate_table_image.py` で画像生成
3. 出力先: `articles/{category}/{slug}/images/table_*.png`
4. 記事内で `![表タイトル](images/table_*.png)` として参照

### 適用タイミング

| フェーズ | 対応 |
|---------|------|
| article-draft | ドラフト生成時に表があれば画像化 |
| article-critique | 「マークダウン表が残っていないか」をチェック項目に含める |
| revised_draft 生成 | 修正版で表が追加された場合も画像化 |

## ルール2: 根拠データにはソースURLを埋め込む

数値データ・統計・専門家の見解を引用する箇所には、**ソースURLをマークダウンリンクとして埋め込む**こと。

### フォーマット

```markdown
<!-- 文中リンク（推奨） -->
純資産総額が[10兆2,846億円に到達](https://example.com/source)しました。

<!-- 末尾注記 -->
CAPE比率は39.8に達しています（[出典: Motley Fool](https://example.com/source)）。
```

### 必須リンク対象

- 具体的な数値データ（純資産総額、リターン、比率など）
- 統計・調査結果の引用
- 専門家・機関の見解の引用
- 公式発表・プレスリリースの参照

### 適用タイミング

| フェーズ | 対応 |
|---------|------|
| article-research | リサーチノートにソースURLを必ず記録 |
| article-draft | ドラフト生成時に引用箇所へリンク埋め込み |
| article-critique | 「主要データにURLリンクがあるか」をチェック |
| revised_draft 生成 | 不足しているリンクを補完 |

## ルール3: チャートは `/generate-chart-image` で画像化する

記事内でデータの可視化（折れ線グラフ、棒グラフ、散布図、ヒートマップ等）が必要な場合、**`/generate-chart-image` スキルでPNG画像を生成**し、画像参照で埋め込むこと。

### 手順

1. チャートデータのJSON定義またはプリセットを用意
2. `/generate-chart-image` スキルで matplotlib + seaborn レンダリング
3. 出力先: `articles/{category}/{slug}/images/chart_*.png`
4. 記事内で `![チャートタイトル](images/chart_*.png)` として参照

### 使用場面

- 時系列データの推移（株価、純資産総額、リターン推移など）
- 比較データの可視化（パフォーマンス比較、資産配分の円グラフなど）
- 週次レポートのチャート生成
- 資産シミュレーションのグラフ作成

### 適用タイミング

| フェーズ | 対応 |
|---------|------|
| article-research | 可視化すべきデータを特定 |
| article-draft | ドラフト生成時にチャート画像を生成・埋め込み |
| article-critique | 「チャートで表現すべきデータがテキストのままになっていないか」をチェック |
| revised_draft 生成 | 不足しているチャートを補完 |

## チェックリスト（批評・最終確認用）

- [ ] マークダウン表が記事内に残っていないか（全て画像化済みか）
- [ ] 主要な数値データにソースURLが埋め込まれているか
- [ ] URLリンクは有効か（壊れたリンクがないか）
- [ ] チャートで表現すべきデータがテキストのままになっていないか
- [ ] 画像ファイル（表・チャート）が `images/` ディレクトリに存在するか
