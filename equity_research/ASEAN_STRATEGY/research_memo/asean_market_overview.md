## 3. インドネシア MSCI格下げリスク — Deep Dive

> **Primary Source**: CLSA "Indonesian market (At a crossroads)"

### 3-1. 格下げ警告の理由
- MSCIインドネシア指数組入れ企業における**株式保有の不透明性**
- **協調取引**（coordinated trading）への懸念
- 一部企業のフリーフロートの実態が不明確

### 3-2. 定量的インパクト（格下げ実現時）

| 項目 | インパクト |
|------|----------|
| 資金流出 | 最大**116億USD**の海外資金流出 |
| 直後の影響 | 警告発表3日間で**7.3億USD**流出、JCI **-7.2%** |
| 通貨 | ルピア下落圧力 |
| 金利 | リスクプレミアム上昇 → 国債利回り上昇 → 金融緩和停止 |
| 格付け | ソブリン格付け引き下げの連鎖リスク |

### 3-3. 規制当局の対応策

| 施策 | 詳細 |
|------|------|
| 最低浮動株比率 | IPO企業の最低浮動株 **7.5% → 15%** へ引き上げ |
| UBOデータ開示 | 上位100社（IDX100）の実質的支配者データをMSCIに提出 |
| 開示期限 | **2026年3月**までに浮動株透明性問題を解決 |
| 市場下支え | 年金・保険の株式投資上限 AUM **8% → 20%**（約**85億USD**の流入ポテンシャル） |

### 3-4. 格下げ回避の可能性と時間軸
- **MSCIの期限**: **2026年5月**までに意義ある進展を要求
- **CLSAの見解**: 株式保有データの透明性確保は「比較的単純なプロセス」
- 当局の厳格な規則適用（上場廃止等のペナルティ含む）があれば、期日までの要件クリアは**十分に可能**

## 4. セルサイド間のコンセンサス vs 相違点

### コンセンサス（一致する見方）
- マレーシア・シンガポール・ベトナムに対して**強気**
- AI/データセンター投資がASEAN横断の構造的テーマ
- 金融緩和サイクルが域内株式市場を下支え

### 相違点・Under-discussed
- インドネシアの評価が分かれる（MSCI格下げリスク vs 政策主導成長）
- タイの政治リスクの織り込み度合い
- ベトナムFTSE格上げの実現時期の不確実性

## 5. バイサイド独自の論点（Phase 7視点）

1. **インドネシアMSCI格下げ**: 最大のテールリスク。116億USDの流出は域内ファンドフロー全体に波及する可能性。2026年3-5月が分岐点。
2. **ベトナムFTSE格上げ**: 50-60億USDのパッシブ流入は大きいが、実現時期の不確実性が高い。TCB/HPG/FPTがベネフィシャリー。
3. **データセンター投資のピークアウトリスク**: ハイパースケーラーの投資計画変更が域内FDIに与える影響は過小評価されている可能性。
4. **非対称リスク**: フィリピン（PER 9x）の割安感は魅力的だが、汚職調査リスクが下値リスクを膨らませている。
