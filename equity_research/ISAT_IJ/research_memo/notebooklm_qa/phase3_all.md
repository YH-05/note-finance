# Phase 3: Company Deep Dive — NotebookLM Q&A

## 3-2. ARPU分析

### Q1: Blended/Prepaid/Postpaid ARPUの四半期推移
- **2Q24:** Blended 約38,400 IDR / Prepaid 約37,500 IDR / Postpaid 約98,700 IDR
- **3Q24:** Blended 約37,200 IDR / Prepaid 約36,400 IDR / Postpaid 約94,100 IDR
- **2Q25:** Blended 約39,000 IDR / Prepaid 約37,000 IDR / Postpaid 約103,000 IDR
- **3Q25:** Blended 約40,000 IDR / Prepaid 約39,000 IDR / Postpaid 約91,000 IDR
- **4Q25:** Blended 約44,000 IDR（Prepaid/Postpaid内訳は未開示）
- Prepaid単価が段階的に底上げ、Postpaid顧客が収益を牽引し、4Q25にBlended ARPUが過去最高の44,000 IDRへ急成長

### Q2: ARPUの変動要因
1. **料金プランの適正化（実質的な値上げと割引廃止）:** スターターパック最低価格を35,000 IDRへ引き上げ、30,000 IDR未満プラン廃止。月額プラン有効期間を30日→28日に短縮
2. **AIを活用したパーソナライゼーション:** ハイパー・パーソナライゼーションが成功。自社アプリ利用者のARPUは平均より70%高く、MAUが4,500万人規模に急成長
3. **「質」重視への競争環境変化:** 業界全体が価格競争から利益重視へシフト。「使い捨て（ローテーショナル）」顧客を排除し、優良顧客獲得に集中。「IM3 Platinum」導入や国際ローミング増もARPU押し上げ要因

### Q3: Data ARPU vs Voice ARPUの構成比とトレンド
- 2025年度実績: データ収入 約43兆3,420億 IDR vs 音声収入 わずか4,920億 IDR
- モバイル収入の**約99%がデータ通信**、音声構成比は**約1%**まで低下
- 音声収入は前年比**26.4%減**と急速縮小
- データ収入も単価低下等で前年比1.9%微減だが、収益の絶対的な柱

## 3-3. データ収益化

### Q4: Data trafficと1加入者あたりデータ使用量の推移
- **総データ通信量:** 2022年12,561 PB → 2023年14,417 PB → 2025年17,655 PB
- **1加入者あたりデータ使用量:** 2024-2025年にかけて毎月約19GB〜22GB
- 高画質動画ストリーミングやモバイルゲームの需要拡大が牽引

### Q5: Data yield（IDR/GB）のトレンド
- 長期的に一貫して低下トレンド
- 2017年: 1GBあたり**13,348 IDR** → 2021年: **3,421 IDR** → 2024-2025年: **約2,300〜2,800 IDR**
- 低下要因: ネットワーク高度化（4G/4G+）によるコスト効率向上、大容量プランの安価提供
- 経営陣は低下をネガティブに捉えず：1GBあたり単価低下でもデータ消費量増加でARPU・通信収益は上昇

### Q6: データ収益化戦略に関する経営陣コメント
- **AI Northstar戦略:** データ収益化をAIビジネスラインの一つとして明確に位置づけ
- **ハイパー・パーソナライゼーション:** AIによるCVM（顧客価値管理）で「n=1」最適提案、無駄な割引排除で単価向上
- **自社アプリでのクロスセル:** MAU4,500万人超の基盤を活かし、デジタルサービスやFTTHを追加販売

## 3-4. インフラ・ネットワーク

### Q7: 基地局数とカバレッジの推移
- **総基地局数:** 2024年末 約25万局 → 2025年末 27万8,173局
- **4G基地局:** 2025年末 21万3,898局
- **5G基地局:** 2024年末 107局 → 2025年末 **6,872局**（爆発的増加）
- **人口カバー率:** 92%到達。新たに1,000万人以上の人口と16,000以上の村落へサービス提供

### Q8: タワー保有数とリース比率・戦略
- 自社保有とリースの正確な比率は未開示だが、大部分はリースに移行
- 基本方針: **「アセット・ライト（Asset-right）」戦略**
- 2019-2023年に8,300基以上のタワーを売却（セール・アンド・リースバック）
- 2023年にMitratel等へ1,630基売却、10年間リースバック契約
- 売却資金をネットワーク拡充に充当し、負債を抑えながら品質強化

### Q9: ファイバー敷設距離とFTTH展開計画
- **ファイバー総延長:** 20万6,275km（別会社切り出し検討分は約8万6,000km）
- **FTTH現在の規模:** ホームパス150万〜200万、アクティブ顧客30万人超（約34万5,400人）
- **短期目標:** 年末までに顧客数40万人
- **中長期目標（2028年）:** 顧客数200万人、固定BB市場シェア8〜10%
- **戦略:** 月額約25万 IDRの中〜高所得者層ターゲット

### Q10: ネットワーク品質の第三者評価
- **Opensignal:** ネットワーク・カバレッジ・エクスペリエンスでインドネシア第2位
- **Opensignal/Tutela:** 「最も一貫したモバイル体験（品質の一貫性）」を提供、「ゲーミング体験」で第1位
- 農村部やジャワ島外でモバイルネットワーク体験が大幅改善

## 3-5. B2B・デジタル事業

### Q11: B2B/ICT事業の売上規模と成長率
- **成長率:** 2023→2024年にかけて**約12%**の高い伸び
- **売上規模（MIDI部門全体）:** 2024年度 約7兆9,870億 IDR → 2025年度 **約8兆3,450億 IDR**
- 利益率（EBITDAマージン）のパーセンテージはB2Cより低いが、純利益への絶対額での貢献が大きい

### Q12: デジタルサービスの展開状況と収益貢献度
- **消費者向け（B2C）:** myIM3/bima+基盤に電子マネー「IMKas」、小口融資「UCan」、ゲーム等を展開。MAU 4,120万人。デジタルサービス収益はコア通信売上の**25%以上**（2023年約18%から急拡大）。アプリ利用者ARPUは平均より**70%高い**
- **法人向け（B2B）:** NVIDIAとの「GPU-as-a-Service」、Google/Ciscoと連携した主権型クラウド/セキュリティ事業。MIDI部門売上は全社売上の**約15%**。GPUaaS単体で4Q25に2,000万ドルの収益創出

### Q13: B2B事業のマージンプロファイル
- B2C事業: EBITDAマージン約55%だが、CapEx負担大で純利益への反映は8%〜11%
- B2B事業: EBITDAマージンはB2Cより低いが、純利益への貢献度（利益変換率）ははるかに高い
- 「絶対額としての純利益」への貢献が大きい
